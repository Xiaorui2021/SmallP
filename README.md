# SmallP
SmallP is designed as auxiliary software with the goal of enhancing the sensitivity and intelligence of disulfide-bonded peptide analysis through collaboration with diverse peptide search tools. SmallP predominantly includes the following modules: fast_stand, SignalP_trans, mgf_cal, Peaks_cal, pfind_cal, and sequence match. The functionalities of each module are detailed as follows:
fast_stand: Streamlines the processing of protein sequences in FASTA files;
SignalP_trans: Summarizes and transforms protein sequences processed through Trinity into Excel results;
mgf_cal: Implements single isotope mass correction on raw mass spectrometry data files;
Peaks_cal: Standardizes the identification results derived from Peaks software, encompassing disulfide-bonded filtering rules and data normalization;
pfind_cal: Standardizes the identification outcomes produced by pfind software, covering disulfide-bonded filtering rules and data normalization;
sequence-match: Aligns peptides identified in distinct groups (RA and RAE) with those identified by Nat.
